<h2><a href="https://www.geeksforgeeks.org/problems/partitions-with-given-difference/1">Partitions with Given Difference</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p dir="ltr"><span style="font-family: 'andale mono', monospace; font-size: 14pt;">Given an array<strong><strong> arr[]</strong></strong> and an integer <strong><strong>diff</strong></strong>, count the <strong><strong>number of ways</strong></strong> to partition the array into two subsets such that the difference between their sums is equal to <strong><strong>diff</strong></strong>.</span></p>
<p dir="ltr"><span style="font-family: 'andale mono', monospace; font-size: 14pt;"><strong><strong>Note:</strong></strong>&nbsp;A partition in the array means dividing an array into two subsets say S<sub>1</sub>&nbsp;and S<sub>2</sub>&nbsp;such that the union of S<sub>1</sub>&nbsp;and S<sub>2</sub>&nbsp;is equal to the original array and each element is present in only one of the subsets.</span></p>
<p><span style="font-size: 14pt; font-family: 'andale mono', monospace;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 14pt; font-family: 'andale mono', monospace;"><strong>Input: </strong>arr[] = [5, 2, 6, 4], diff = 3
<strong>Output: </strong>1
<strong>Explanation: </strong>There is only one possible partition of this array. Partition : [6, 4], [5, 2]. The subset difference between subset sum is: (6 + 4) - (5 + 2) = 3.</span></pre>
<pre><span style="font-size: 14pt; font-family: 'andale mono', monospace;"><strong>Input:</strong> arr[] = [1, 1, 1, 1], diff = 0 <br><strong>Output:</strong> 6 <br><strong>Explanation: </strong>We can choose two 1's from indices [0,1], [0,2], [0,3], [1,2], [1,3], [2,3] and put them in sum1 and remaning two 1's in sum2.<br>Thus there are total 6 ways for partition the array arr. <br></span></pre>
<pre><span style="font-size: 14pt; font-family: 'andale mono', monospace;"><strong>Input:</strong> <span class="GFGEditorTheme__textItalic">arr[] = [</span>3, 2, 7, 1]<span class="GFGEditorTheme__textItalic">, diff = 4</span>  </span><br><span style="font-family: 'andale mono', monospace; font-size: 14pt;"><strong>Output:</strong> 0<br><strong>Explanation: </strong>There is no possible partition of the array that satisfy the given difference.<strong> </strong></span></pre>
<p><span style="font-size: 14pt; font-family: 'andale mono', monospace;"><strong>Constraint:</strong></span><br><span style="font-size: 14pt; font-family: 'andale mono', monospace;">1 ≤ arr.size()&nbsp;≤&nbsp;50<br>0 ≤ diff ≤ 50<br>0 ≤ arr[i] ≤ 6</span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Dynamic Programming</code>&nbsp;<code>Arrays</code>&nbsp;
